# Life Expectancy LightGBM Deployment

This repo contains code to train a LightGBM regression model on `Life Expectancy Data.csv`, save it, and serve it via FastAPI.

## 1) Train & save model

1. Place `Life Expectancy Data.csv` in the repository root (same level as `train_save_model.py`) or set `DATA_PATH` env variable.
2. Run:

```bash
python train_save_model.py
```

3. The model will be saved to `models/lightgbm_life_expectancy.pkl`.

## 2) Run locally

```bash
uvicorn app:app --reload --host 0.0.0.0 --port 8000
```

- `GET /` returns model status and feature set
- `POST /predict` with JSON `{ "data": { ... } }` returns prediction

Example:

```bash
curl -X POST "http://localhost:8000/predict" -H "Content-Type: application/json" -d '{"data": {"Year": 2000, "Status": 0, ...}}'
```

## 3) Deploy to Render

### 3.1) Render (standard Python mode)

1. Connect repository in Render dashboard.
2. `Branch`: `main`
3. `Root Directory`: `.`
4. `Plan`: `Starter` (or higher)
5. `Environment`: `Python`
6. Build Command:

```bash
pip install --upgrade pip && pip install -r requirements.txt && python train_save_model.py
```

7. Start Command:

```bash
uvicorn app:app --host 0.0.0.0 --port 8000
```

8. Environment variables (optional):

- `DATA_PATH=Life Expectancy Data.csv`
- `MODEL_PATH=models/lightgbm_life_expectancy.pkl`
- `MODEL_DIR=models`

### 3.2) Render (Docker deploy mode)

1. On Render, create a new **Web Service**.
2. Select **Docker** as the environment.
3. `Branch`: `main`
4. `Root Directory`: `.`
5. Leave `Dockerfile` path at default (`./Dockerfile`).
6. No build command needed; Render executes Dockerfile.
7. Start command is configured in Dockerfile (`uvicorn app:app --host 0.0.0.0 --port 8000`).
8. Verify model files in repo: `models/lightgbm_life_expectancy.pkl` is already generated by `train_save_model.py` and committed.

Use this one-liner on a local machine before pushing if you need to regenerate the model:

```bash
python train_save_model.py
```

9. Optional env vars in Render:

- `MODEL_PATH=models/lightgbm_life_expectancy.pkl`
- `MODEL_DIR=models`
- `DATA_PATH=Life Expectancy Data.csv`
