Streamlit Cloud — minimal checklist

1) Main file = streamlit_app.py (NOT app.py).

2) requirements.txt on GitHub = copy requirements-streamlit-cloud.txt
   (streamlit, pandas, numpy, lightgbm, joblib — no sklearn, no seaborn).

3) Python 3.12 or 3.11 in Advanced settings when creating the app.

4) Commit models/lightgbm_life_expectancy.pkl (run train_save_model.py locally first).

5) Streamlit app must use the same GitHub repo you push to (not an old ml-model repo).
